## Instruction the play the game again

#### Tic-Tac-Toe
1. Select Player vs Player or Player vs Computer based the icon which is selected
    * Player vs Player
      Two players have the ability to pass the remote and play tic-tac-toe with each other.
    * Player vs Computer<br>
    <img src="https://drive.google.com/uc?export=view&id=1wH1U1EPgtICfDrN1kHymeQuZC6uG54wh" width="300" height="400"><br>
      1. Select a difficulty (Easy or Medium)
          * Easy: Computer places randomly
          * Medium: Computer makes some random moves but also takes win opportunities and blocks wins for the opponent
2. Play until there is a win or tie.
    * Press the back button to go to the game libraries
    * Press OK to play the game again

#### Elevens
1. The position of the player will automatically be the middle card whenever the game is opened
2. The user has to be concentrated and memorize the position they are on the board
    * The player can double check by selecting the card to find out where their cursor is
3. In order to win the game, the user need to select either:
    * two cards that can be added up to elevens(Ace is 1, 2 is 2, ..., 10 is 10)
    * three face cards: jack, queen, king.
4. The user needs to determine when to reset the game if they can't find any required combinations
    * A Play again button is available when the user moves the cursor to it and clicks **OK**
5. To win the game, the player must be able to remove all the cards visible on screen.

#### Flappy Bird
1. On the starting screen, press **OK** on the remote to press the "Start" button displayed on the screen
    * The bird will start accelerating towards the ground, so you have to continuously press **OK** to keep the bird jumping
2. Get the bird to fly through the gap between the pillars
    * When the bird went through, the user score at the top of the screen will be incremented by one
    * Try to earn as many points as you can
3. If the bird touches one of the green pillars at all
    * The game will stop
    * The player will be able to see the score they earned
    * A play again button will pop up
    * Press **OK** to play again
4. The game will keep playing continuously until you lose the game
5. If you want to leave the game, click **BACK** on the remote
